1.Object (5 way we can define object in js)
--------------------------------------------
Object is the instance of class.It means it comes with2  key pair key and value

1-Object Literal:

The most common way to create an object is by using the object literal notation:
var person = {
  name: "John",
  age: 30,
  isStudent: false
};

2: Object Constructor:

You can use the Object constructor to create an empty object and then add properties and methods later:
var person = new Object();
person.name = "John";
person.age = 30;
person.isStudent = false;

3: Object.create():

You can use the Object.create() method to create a new object, with an optional prototype object:
var personPrototype = {
  greet: function() {
    console.log("Hello!");
  }
};

var person = Object.create(personPrototype);
person.name = "John";
person.age = 30;
person.isStudent = false;

4: Constructor Function:

You can define a constructor function and use the new keyword to create instances of objects:
function Person(name, age, isStudent) {
  this.name = name;
  this.age = age;
  this.isStudent = isStudent;
}
var person = new Person("John", 30, false);

5: Class (ES6 and later):

With the introduction of ES6, you can use the class syntax to define objects with constructors and methods:
class Person {
  constructor(name, age, isStudent) {
    this.name = name;
    this.age = age;
    this.isStudent = isStudent;
  }

  greet() {
    console.log("Hello!");
  }
}
var person = new Person("John", 30, false);
